В архиве находится исходный код и файл source.url с ссылкой на репозиторий с этим же исходным кодом.
Также присутствует собранный apk: app-debug-yandex-builuk.apk

Работа выполнена с помощью Jetpack Compose без использования сторонних библиотек.

Некоторые моменты из моей реализации:

Верхняя панель:
1) При одиночном клике на инструмент добавляется слой с дефолтным сэмплом для данного иструмента.
2) При долгом нажатии на инструмент открывается меню с доступными сэмплами.
3) При долгом нажатии на сэмпл из меню он проигрывается.
4) При одиночном клике на сэмпл из меню добавляется слой с данным сэмплом.

Визуализация аудио:
1) Работает только во время записи микрофона или всего трека.
2) Деления закрашиваются в зависимости от громкости в текущий момент.

Кнопка записи трека:
1) При завершении записи, автоматически предлагается поделиться пользователю звуковым файлом с записанным треком.
